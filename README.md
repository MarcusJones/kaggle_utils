# 
Utilities for ML, AI, and kaggle competitions





Workflow for utils
1. Git clone
1. pip install -e PATH_TO_THIS_REPO

For your changes
1. git commit and push
1. Note that you will need to re-import the changes into memory! See `autoreload`

For incoming changes
1. git pull
1. Note that you will need to re-import the changes into memory! See `autoreload`
